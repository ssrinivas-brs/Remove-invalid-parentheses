Remove Invalid Parentheses

This code defines a function removeInvalidParentheses that takes a string s as input and returns an array of valid strings with the minimum number of removals. The isValid function checks if a given string is valid with balanced parentheses. The BFS approach is used to explore all possible combinations by removing one parenthesis at a time and checking if the resulting string is valid. The visited set is used to avoid duplicates in the queue
